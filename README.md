## Installation

Download the latest release for your platform from the [releases page](https://github.com/astrum-chat/astrum/releases/latest).

<details>
<summary><strong>Build from source</strong></summary>

### Prerequisites

- Rust toolchain (1.75+)

### Build

```bash
git clone https://github.com/astrum-chat/astrum.git
cd astrum
cargo build --release
```

### Run

```bash
cargo run --release
```

</details>
